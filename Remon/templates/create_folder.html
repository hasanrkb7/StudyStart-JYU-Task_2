Method: This uses a POST request with JSON data to create a new folder

Create New Folder

POST Request Details:

  • URL: https://api.dropboxapi.com/2/files/create_folder_v2
  • Method: POST
  • Headers:
    • Authorization: Bearer [access_token]
    • Content-Type: application/json
  • Body: JSON with path and autorename settings
  • Destination: /StudyStartJYU_folders/[folder_name]
Note: Folders will be created in a directory called "StudyStartJYU_folders" in your Dropbox. The autorename feature is enabled, so if a folder with the same name exists, it will be automatically renamed.
