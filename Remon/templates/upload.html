Upload File to Dropbox

POST Request Details:

  • URL: https://content.dropboxapi.com/2/files/upload
  • Method: POST
  • Headers:
    • Authorization: Bearer [access_token]
    • Content-Type: application/octet-stream
    • Dropbox-API-Arg: JSON metadata
  • Body: Binary file data
  • Destination: /StudyStartJYU_uploads/[filename]
Note: Files will be uploaded to a folder called "StudyStartJYU_uploads" in your Dropbox. The autorename feature is enabled, so if a file with the same name exists, it will be automatically renamed.
